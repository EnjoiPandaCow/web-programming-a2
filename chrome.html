Google Chrome

Google Chrome is a free-ware browser developed by Google using the WebKit layout engine. It was first release on Microsoft Windows operating system on September 2, 2008 in 43 different languages in beta version. It was written in C++, Assembly, Python, and JavaScript. Now a days it is available in 50 languages

Advantages

  • Fast browsing performance
  • Easy to control all your tabs
  • More dynamic home page for common used search engines and bookmarks
  • Faster and won’t slow down on your browsing through internet
  • Built-in Flash and PDF Support
  • Has sync capabilities that allow you to access your customized browser from any computer
  • Makes searching a lot easier
  • Easy to use
  • Security level is very high
  • Free to download at no cost

Disadvantages

  • Lack of parental control
  • Minor site incompatibilities
  • Graphics hardware acceleration does not’t work with all graphics cards
